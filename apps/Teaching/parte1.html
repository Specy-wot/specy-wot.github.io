HTML
Html usa una struttura a tag ad albero, un po' come un grafo, ogni elemento (tranne il nodo padre, cioè il tag html) ha un nodo padre, e può avere uno o più nodi figli.

Il nodo padre è chiamato "parent" mentre quello figlio, "children".

Lo scopo di html è di creare una struttura tramite l'utilizzo dei vari tag disponiili.

Ogni tag ha caratteristiche diverse uno dall'altro che lo rendono più adatto di altri in certe situazioni. Quelli che vedremo in questo tutorial sono:

  • div
  • span
  • input
  • button
  • table
  • tr
  • td
  • img
  • a
  • ul / ol
  • li
  • select
  • option
div

span
Il tag span fa poco che nulla da solo, è un semplice contenitore di testo che può poi essere modificato tramite css, al suo posto si può anche usare una div 

    
        Test span
input
Il tag input è uno dei tag più importanti di HTML5, esso può compiere tantissimi ruoli e ne esistono di molti tipi per i vari scopi necessari.

Tutti i tag input hanno bisogno di avere un "type" , il type indica il tipo di input che può per esempio essere: button, checkbox, text, file, password, range etc

Una cosa che hanno tutti in comune però è l'attrubuto "value" , value indica il valore attuale dell'input, che in base al tipo sarà di diverso formato.

Per più informazioni visita questo sito
button
Il tag button è un bottone, con questo bottone si possono eseguire funzioni javascript. Questo è un buon momento per introdurre gli "event listener" di html/Javascript.

Essi servono ad "ascoltare gli eventi", questi eventi sono per esempio un click, puntatore sopra l'elemento, etc...

Gli event listener hanno ogniuno un proprio nome, di solito sono chiamati "on + tipo evento" come per esempio: "onClick" e possono essere aggiunti a tutti i tipi di tag.

Qui ci sono tutti i tipi di eventi
table
Il tag table serve a contenere una griglia di dati, è formata da colonne e righe, righe chiamate "tr" e colonne chiamate "td / th".

In HTML5 si sono aggiunti 3 tipi di table interni: thead (titolo), tbody (corpo), tfoot (finale)

Una table può contenere una o più righe e le righe possono contenere una o più colonne Le table sono consigliate SOLO E SOLAMENTE per mostrare dati, NON per creare layout e pezzi di sito. Per fare layout si invece usano le div.
tr
Il tag tr è una riga della table, essa poi contiene una o più colonne (td / th)
td / th
I tag td / th sono le colonne di una table e si trovano all'interno di una riga.
th è di solito usato come "header", cioè il titolo di una colonna, come "nome".
td è l'elemento vero e proprio della colonna

img
Il tag img è utilizzato per aggiungere un immagine/gif alla pagina html. Esso usa l'attributo "src" (source) per indirizzare alla cartella della foto
a
Il tag a viene utilizzato per reindirizzare la pagina ad un altro sito / pagina / file. Utilizza l'attributo "href" (Hypertext REFerence) per reindirizzare ad un altra risorsa.

ul / ol
I tag ul e ol sono tag per liste, essi hanno lo scopo di raggruppare tante linee in una lista ordinata o meno.

ul stà per unordered list, utilizza un pallino (bullet) per ogni linea.
ol stà per ordered list, utilizza un numero.

Ogni linea è rappresentata dal tag "li"

select
Il tag select è un elemento che si usa quando si deve selezionare un elemento all'interno di una lista di oggetti, ogni opzione è poi un tag a sua volta chiamato option
option
Il tag option è un opzione che và all'interno del tag select, esso può anche avere 2 attributi chiamati: "selected" e "disabled", selected vuol dire che quello è l'elemento che verrà selezionato come primo valore appena si entra nella pagina, disabled vuol dire che non si può selezionare quel valore
Attributi
Gli attributi sono parte degli elementi html che rappresentano una caratteristicha.
Quelli più usati e che sono validi per tutti gli elementi sono:

class: serve per specificare una classe css per aggiungere lo stile all'elemento

style: serve per aggiungere stile in un elemento senza usare una classe (NON DA USARE AL POSTO DI CLASS)

id: serve per identificare un elemento all'interno della pagina, questo può poi essere usato con javascript per la manipolazione degli elementi
